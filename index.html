pandas入门

发表于 2017-05-20   |   分类于 Python ， pandas

Series

Series是一种类似于一维数组的对象，它由一组数据以及一组与之相关的数据标签（索引）组成

DataFrame

表格型的数据结构，它含有一组有序的列，每列可以是不同的值类型；DataFrame既有行索引也有列索引，可以被看做是Series组成的字典。

索引

1. 重新索引

方法： reindex(new_index, fill_value=?, method=”, …)

参数列表：

参数 说明
index 用作索引的新序列
method 填充方法，参看下表
fill_value 重新索引时，填充值
limit 最大填充量
level 在MultiIndex的指定级别上匹配简单索引，否则选择其子集
copy 默认是True，无论如何都复制；如果为False，相同值不再复制

method方法：

  • ffill | pad： 前向填充（或搬运）值
  • bfill | backfill: 后向填充（或搬运）值

2. 删除索引

drop()

3. 索引、选取和过滤

可以用索引、下标，切片（末端是包含的）

4. 算术运算和数据对齐

pandas最重要的一个功能就是可以对不同索引的对象进行算术运算。

在将对象相加时，如果存在不同的索引对，则结果的索引就是该索引对的并集。

运算方法：

  1. add; sub; div; mul, 例：obj1.add(obj2, fill_value=0)
  2. 可以在调用上述方法时指定默认值，也可以得到结果后重新使用reindex填充数据

5. 函数运算

  1. apply
  2. applymap

6. 排序与排名

  1. 排序：对行或者列索引进行排序，
    • sort_index(axis=1, ascending=False, by=’index’, )
  2. 排名：对数据继续排名，返回的排名的值
    • rand(ascening=False, method=)
    • method: [‘max’, ‘min’, ‘first’, ‘averange’], 排名时用于破坏平级关系的选项

7. 有重复值的索引

取值： obj.index.is_unique
查找：重复索引查询，获得多行内容

处理缺失数据

pandas的设计目标之一就是让缺失数据的处理任务尽量轻松

NA处理方法

方法 说明
dropna 根据各标签的值中是否存在缺失数据对轴标签进行过滤，可通过阈值调节对缺失值的容忍度
fillna 用指定值或差值方法填充缺失数据
isnull 返回一个布尔值对象
notnull isnull的否定式

dropna方法

  1. how=’all’: 只丢弃群为NA的行
  2. thresh=2: 过滤count<thresh的行

fillna

  1. df.fillna(0): 填充指定值
  2. df.fillna({1:0.5, 3:-1}): 指定不同列不同填充值

参数

  1. value：用于填充的标量值或者字典对象
  2. method：插值方式
  3. axis：填充的轴，默认是0
  4. inplace：修改调用者对象而不产生副本
  5. limit： 可以连续填充的最大数量

AOP

注：面向切面编程（Aspect Oriented Programming）

为什么使用切面

我们软件开发时经常提一个词叫做“业务逻辑”或者“业务功能”，我们的代码主要就是实现某种特定的业务逻辑。但是我们往往不能专注于业务逻辑，比如我们写业务逻辑代码的同时，还要写事务管理、缓存、日志等等通用化的功能，而且每个业务功能都要和这些业务功能混在一起，痛苦！所以，为了将业务功能的关注点和通用化功能的关注点分离开来，就出现了AOP技术。这些通用化功能的代码实现，对应的就是我们说的切面（Aspect）。

Spring学习笔记-IoC

发表于 2016-12-03   |   分类于 Java ， Spring

IoC

控制反转IoC(Inversion of Control)，是一种设计思想，DI(依赖注入)是实现IoC的一种方法，也有人认为DI只是IoC的另一种说法。没有IoC的程序中我们使用面向对象编程对象的创建与对象间的依赖关系完全硬编码在程序中，对象的创建由程序自己控制，控制反转后将对象的创建转移给第三方，个人认为所谓控制反转就是：获得依赖对象的方式反转了。

应用控制反转后，当对象被创建时，由一个调控系统内的所有对象的外界实体将其所依赖的引用传递给它，即依赖被注入到对象中。所以，控制反转是冠以一个对象如何获取它所依赖的对象的引用，在这里，反转指的是责任的反转。
