Python 数据结构

发表于 2016-10-08   |   分类于 Python

列表

1. 基本函数

  • append(x)：列表结尾插入数据
  • extend(L)：一个列表添加到另外一个列表后面
  • insert(i, x)：指定位置插入数据
  • remove(x)：删除x元素
  • pop([i=len(a)])：删除位置i的元素，默认最后一个
  • index(x)：返回第一个值为x元素的索引
  • count(x)：统计x出现的次数
  • sort(cmp=None, key=None, reverse=False)：对列表进行排序
  • reverse()： 倒排

2. 链表可以直接当成堆栈使用

3. 队列可以使用链表实现，但是推荐使用[collections.deque]

函数式编程

1. filter

filter(function, sequence) 返回一个sequence(序列)，function用来判断，如果为True，返回==该数据==（即参数）。如果该 序列 (sequence)是一个 str, unicode 或者 tuple，返回值必定是同一类型，否则，它总是 list。

4. 列表推导式

列表推导式为从序列中创建列表提供了一个简单的方法。普通的应用程序通过将一些操作应用于序列的每个成员并通过返回的元素创建列表，或者通过满足特定条件的元素创建子序列。

5. zip

定义：zip([seql, …])接受一系列可迭代对象作为参数，将对象中对应的元素打包成一个个tuple（元组），然后返回由这些tuples组成的list（列表）。若传入参数的长度不等，则返回list的长度和参数中长度最短的对象相同。

del语句

使用del可以删除指定索引的数据，还可以删除切片或者整个列表

元组（tuple）

  • 输入时可以没有括号，输出时总是有括号的
  • 元组是不可变的，而列表是可变的

循环技巧

  • enumerate 遍历列表，可以获得下表和内容
  • iteritems 遍历字典，获得k, v
  • zip 同时循环两个或更多的序列
  • reversed 逆序列表
