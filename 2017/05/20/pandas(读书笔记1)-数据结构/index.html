pandas(读书笔记1)-数据结构

发表于 2017-05-20   |   分类于 Python ， pandas

pandas 是基于NumPy构建的，一个开源的数据结构化和分析工具。

Series

基本知识

Series是一种类似于一维数组的对象，它由一组数据以及一组与之相关的数据标签（索引）组成，索引在左边，值在右边，默认索引为0到N-1的整数型索引。

使用方法示例

DataFrame

基础知识

  1. DataFrame是一个表格型的数据结构，它含有一组有序的列，每列可以是不同的值类型；
  2. DataFrame既有行索引也有列索引，可以被看做是Series组成的字典。
  3. DataFrame中的数据是以一个或多个二维块存放的

创建DataFrame数据

示例

构造方法

类型 说明
二维ndarray 数据矩阵，还可以传入行标和列标
由数组、列表或元祖组成的字典 每个序列会变成DataFrame的一列，所有序列的长度必须相同
NumPy的结构化/记录数组 类似于”由数组组成的字典”
由Series组成的字典 每个Series成为一列。如果没有指定索引，则各Series的索引会被合并成结果的行索引
由字典组成的字典 各内层字典会成为一列，键会被合并成结果的行索引
字典或Series列表 各项将会成为DataFrame的一行，字典键或Series索引的并集将会成为DataFrame的列标
由列表或元祖组成的列表 类似于”二维ndarray”
另一个DataFrame 对象拷贝，可指定新的索引
NumPy的MaskedArray 类似于”二维码ndarray”的情况，只是掩码值在结果DataFrame会变成NA/缺失值

索引

1. 重新索引

方法： reindex(new_index, fill_value=?, method=”, …)

参数列表：

参数 说明
index 用作索引的新序列
method 填充方法，参看下表
fill_value 重新索引时，填充值
limit 最大填充量
level 在MultiIndex的指定级别上匹配简单索引，否则选择其子集
copy 默认是True，无论如何都复制；如果为False，相同值不再复制

method方法：

  • ffill | pad： 前向填充（或搬运）值
  • bfill | backfill: 后向填充（或搬运）值

4. 算术运算和数据对齐

pandas最重要的一个功能就是可以对不同索引的对象进行算术运算。

在将对象相加时，如果存在不同的索引对，则结果的索引就是该索引对的并集。

运算方法：

  1. add; sub; div; mul, 例：obj1.add(obj2, fill_value=0)
  2. 可以在调用上述方法时指定默认值，也可以得到结果后重新使用reindex填充数据

5. 函数运算

  1. apply
  2. applymap

6. 排序与排名

  1. 排序：对行或者列索引进行排序，
    • sort_index(axis=1, ascending=False, by=’index’, )
  2. 排名：对数据继续排名，返回的排名的值
    • rand(ascening=False, method=)
    • method: [‘max’, ‘min’, ‘first’, ‘averange’], 排名时用于破坏平级关系的选项

统计计算

pandas 拥有一组常用的数学和统计方法，用户从Series中提取单个值（sum/mean）或从DataFrame的行或列中提取一个Series。

统计方法

方法 说明
count 非NA值的数量
describe 针对列计算汇总数据
min/max 最小值（最大值）
argmin、argmax 最小值（最大值）的索引位置
idxmin、idxmax 最小值（最大值）的索引值
quantile 样本的分位数（0-1）
sum 求和
mean 平均数
meidan 中位数
mad 根据平均值计算平均绝对离差
var 样本值的方差
std 样本值的标准差
skew 偏度（三阶矩）
kurt 峰度（四阶矩）
cumsum 累计和
cummin、cummax 累计最大值和累计最小值
cumprod 累计积
diff 计算一阶差分
pct_change 计算百分数变化

处理缺失数据

pandas的设计目标之一就是让缺失数据的处理任务尽量轻松

NA处理方法

方法 说明
dropna 根据各标签的值中是否存在缺失数据对轴标签进行过滤，可通过阈值调节对缺失值的容忍度
fillna 用指定值或差值方法填充缺失数据
isnull 返回一个布尔值对象
notnull isnull的否定式

dropna方法

  1. how=’all’: 只丢弃群为NA的行
  2. thresh=2: 过滤count<thresh的行

fillna

  1. df.fillna(0): 填充指定值
  2. df.fillna({1:0.5, 3:-1}): 指定不同列不同填充值

参数

  1. value：用于填充的标量值或者字典对象
  2. method：插值方式
  3. axis：填充的轴，默认是0
  4. inplace：修改调用者对象而不产生副本
  5. limit： 可以连续填充的最大数量

常用函数

  1. value_counts: 统计多维数组所有元素出现次数

注意：该函数是Series的方法，而不是DataFrame的方法

  • series.value_counts()
  • df[column_name].value_counts()
  • pd.value_counts(df.values.flatten())
